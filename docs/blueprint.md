# **App Name**: Shabik Labik Digital

## Core Features:

- Secure Mobile Auth: Phone number and password based registration and login flow for quick access to the platform.
- Dynamic Balance Wallet: Real-time user balance tracking with a streamlined UI for viewing total Syr credit.
- Product Sync Tool: Automatic API connection to fetch product lists and pricing directly from provider bridges like Al-Ragheb.
- Multi-Provider Ordering: Capability to order gaming currencies (PUBG, Free Fire) and mobile cash transfers (Syr Cash, MTN Cash) within one view.
- Deposit Notification Portal: Interface for users to submit deposit notifications for manual verification by uploading transaction details.
- Maalam Admin Console: Privileged control panel for administrators to approve/deny credits and manage global pricing variables.
- Smart Support Assistant: Generative AI powered support tool that helps users troubleshoot transfer details and provides natural language responses to transaction queries.

## Style Guidelines:

- Primary Color: Deep Royal Blue (#3b357d), evoking a sense of trust, legacy, and digital expertise.
- Background Color: Ultra-soft Pearl Gray (#f7f7f9), ensuring the content and interactive cards stand out in a light-mode scheme.
- Accent Color: Vibrant Cerulean (#3079ea), utilized for active state buttons, balance highlights, and key navigation calls to action.
- Main font pairing: 'Inter' for crisp UI controls and transactional data, ensuring readability in small mobile containers.
- Minimalist outlined stroke icons for service categories, ensuring consistent line weights for PUBG, MTN, and Siriatel modules.
- A focused card-based grid layout tailored for one-handed mobile use, featuring a prominent hero-section wallet display.
- Micro-transitions for price updates and sliding sheet interactions when users select gaming products or services.
