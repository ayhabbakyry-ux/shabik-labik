
# شبك لبيك الرقمي - دليل النشر والإنتاج (Vercel)

هذا المشروع مجهز ليعمل كمنصة خدمات رقمية متكاملة. لضمان تطبيق الإصلاحات الأخيرة وتصفير الأرصدة، يجب اتباع خطوات النشر التالية على Vercel.

## 1. التحديث التلقائي (Auto-Deploy)
بمجرد حفظ التعديلات في بيئة التطوير، سيقوم مستودع GitHub الحالي باستلام الملفات الجديدة. إذا كان المشروع مربوطاً بـ Vercel، سيبدأ البناء تلقائياً.

## 2. ضبط المتغيرات البيئية (Environment Variables) - خطوة حرجة
بما أننا قمنا بتأمين المفاتيح وإزالتها من الكود الصريح، يجب عليك إضافتها في لوحة تحكم Vercel الخاصة بمشروعك:
1. اذهب إلى **Settings** ثم **Environment Variables**.
2. أضف المتغيرات التالية:
   - **Key**: `ALRAGHEB_TOKEN` | **Value**: `64659dc283eb8ee87192b012aaec33b07d56a00ddf18bdc0`
   - **Key**: `GEMINI_API_KEY` | **Value**: (ضع مفتاح جوجل الخاص بك هنا)
3. اضغط على **Save**.

## 3. إعادة النشر (Redeploy)
إذا قمت بإضافة المتغيرات بعد الرفع، يفضل عمل **Redeploy** لضمان قراءتها:
1. اذهب إلى تبويب **Deployments** في Vercel.
2. اختر آخر عملية بناء واضغط على النقاط الثلاث (...) ثم اختر **Redeploy**.

## 4. الملاحظات الفنية للإصدار الأخير
- **نظام الإحالة**: تم تفعيل كود `ADMEN` وكود المشتركين (آخر 5 أرقام).
- **الحسبة الذكية**: زيادة 0.20 ل.س.ج لكل 10 وحدات رصيد، وزيادة 2 ل.س.ج ثابتة للألعاب.
- **سجل الرصيد**: يظهر الآن الرصيد قبل وبعد العملية بالألوان (أخضر وأحمر) في صفحة الطلبات.
