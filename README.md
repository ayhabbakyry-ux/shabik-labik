
# شبك لبيك الرقمي - دليل النشر الاحترافي (Vercel)

هذا المشروع مجهز ليعمل كمنصة خدمات رقمية متكاملة. لضمان عمل الـ APIs والربط مع متجر الراغب، يجب اتباع الخطوات التالية للنشر على Vercel.

## 1. ربط المشروع بـ Vercel
1. اذهب إلى [Vercel](https://vercel.com) وسجل دخولك باستخدام حساب GitHub.
2. اضغط على **"Add New Project"**.
3. ابحث عن مستودعك: `ayhabbakyry-ux`.
4. اضغط على **"Import"**.

## 2. إضافة المتغيرات البيئية (Environment Variables) - خطوة حرجة
أثناء إعداد المشروع في Vercel، وقبل الضغط على "Deploy":
1. ابحث عن قسم **"Environment Variables"**.
2. في خانة **Key** اكتب: `ALRAGHEB_TOKEN`
3. في خانة **Value** الصق التوكن الخاص بك: `64659dc283eb8ee87192b012aaec33b07d56a00ddf18bdc0`
4. اضغط على **"Add"**.

## 3. النشر النهائي
1. اضغط على زر **"Deploy"**.
2. انتظر دقيقتين حتى يكتمل بناء المشروع.
3. سيعطيك Vercel رابطاً حياً مثل: `shabik-labik.vercel.app`.

## 4. الملاحظات التقنية
- تم تأمين التوكن عبر `process.env`.
- تم ضبط `next.config.ts` ليتجاهل أخطاء البناء الطفيفة لسرعة النشر.
- النظام مجهز ببيانات احتياطية (Fallback) لضمان عدم توقف الواجهة أبداً.
