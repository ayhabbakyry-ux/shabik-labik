
# شبك لبيك الرقمي - دليل النشر والإنتاج (Vercel)

هذا المشروع مجهز ليعمل كمنصة خدمات رقمية متكاملة. لضمان تطبيق الإصلاحات الأخيرة وتصفير الأرصدة، يجب اتباع خطوات النشر التالية على Vercel.

## 1. التحديث التلقائي (Auto-Deploy)
بمجرد حفظ التعديلات في بيئة التطوير، سيقوم مستودع GitHub الحالي (`ayhabbakyry-ux`) باستلام الملفات الجديدة. إذا كان المشروع مربوطاً بـ Vercel، سيبدأ البناء تلقائياً.

## 2. ضبط المتغيرات البيئية (Environment Variables) - خطوة حرجة
في لوحة تحكم Vercel الخاصة بمشروعك:
1. اذهب إلى **Settings** ثم **Environment Variables**.
2. أضف المتغير التالي (إذا لم يكن موجوداً):
   - **Key**: `ALRAGHEB_TOKEN`
   - **Value**: `64659dc283eb8ee87192b012aaec33b07d56a00ddf18bdc0`
3. اضغط على **Save**.

## 3. إعادة النشر (Redeploy)
إذا كانت الواجهة لا تزال قديمة:
1. اذهب إلى تبويب **Deployments** في Vercel.
2. اختر آخر عملية بناء واضغط على النقاط الثلاث (...) ثم اختر **Redeploy**.
3. تأكد من تفعيل خيار **Use existing Build Cache** (أو عطله لضمان بناء نظيف تماماً).

## 4. الملاحظات الفنية للإصدار الأخير
- **تصفير الرصيد**: تم ضبط الرصيد الافتراضي للمستخدمين الجدد ليكون 0 ل.س.ج.
- **إصلاح الواجهة**: تم تنظيف أكواد Tailwind في لوحة الإدارة لمنع التداخل.
- **الربط الحقيقي**: النظام الآن يتصل بمتجر الراغب ويعالج الحقول العربية آلياً.
