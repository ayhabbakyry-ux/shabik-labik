
# شبك لبيك الرقمي - دليل النشر والتشغيل

هذا المشروع مبني باستخدام Next.js ويحتوي على عمليات برمجية (API Routes)، لذا يجب رفعه على استضافة تدعم خوادم Node.js مثل **Vercel** ليعمل بشكل صحيح.

## 1. التشغيل المحلي (Development)
للعودة للعمل على البيئة المحلية:
1. تأكد من أنك في مجلد المشروع الرئيسي.
2. شغل الأمر: `npm run dev`
3. افتح الرابط: `http://localhost:9002`

## 2. خطوات الرفع على Vercel (الحل النهائي لخطأ 404)
منصة GitHub Pages تدعم المواقع الاستاتيكية فقط ولا تدعم الـ API الخاص بنا، لذا اتبع الآتي:
1. قم بإنشاء حساب على [Vercel](https://vercel.com).
2. قم برفع كود المشروع إلى مستودع (Repository) جديد على حسابك في GitHub.
3. في Vercel، اختر "Add New Project" واربط مستودع الـ GitHub الخاص بك.
4. اترك إعدادات البناء كما هي (Default) واضغط **Deploy**.
5. سيعطيك Vercel رابطاً حياً يعمل عليه الموقع والـ API والمنتجات فوراً.

## 3. ملاحظات هامة
- تم دمج التوكن الخاص بمتجر الراغب داخلياً.
- تم ضبط نظام الطوارئ (Fallback) لضمان عدم توقف الواجهة أبداً.
